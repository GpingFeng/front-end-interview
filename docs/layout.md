## CSS 布局

### 盒子模型

-  盒子模型包括：内容（content），填充（padding），边框（border），边界（margin）
- 有两种模型。一种是IE盒子模型、另一种是标准W3c盒子模型。其中IE盒子模型content包括了padding和border
